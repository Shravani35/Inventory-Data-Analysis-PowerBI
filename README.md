# Inventory Demand & Supply Analysis Dashboard

## Tools Used
Power BI | SQL Server | MySQL | DAX | Excel

## Project Overview
Analyzed production inventory dataset to identify demand-supply gaps 
and quantify financial impact across products.

## Key Steps
- Performed ETL operations using SQL Server (null value treatment, 
  duplicate removal, date format standardization)
- Developed DAX measures to calculate business KPIs:
  - Total Profit
  - Total Loss (8M)
  - Average Daily Loss (2.97K)
- Built interactive Power BI dashboard with KPI cards
- Integrated MySQL database with Power BI for live reporting
